# Movemint-Ledger

A foundational financial ledger system built on the Studio Standard v1 principles.

## Overview

Movemint-Ledger provides transparent, reliable, and modular transaction tracking for modern applications. See [OPS_PROJECT.md](./OPS_PROJECT.md) for detailed project vision and scope.

## Getting Started

### Prerequisites

- Node.js 22+
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Testing

```bash
npm run test
npm run lint
```
