## Quick Start

```bash

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start
```

## Endpoints

- **Landing Page**: `/movemint` - Public-facing page explaining the ledger system
- **Ledger API**: `/api/ledger` - REST API endpoint for transaction queries
